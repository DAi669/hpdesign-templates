# cafe-california-open-air 設計ノート

## コンセプト

**California Open Air** — 西海岸×オープンエア感、サーフボードとテラスの光、湘南/海岸エリアにマッチするライフスタイル提案型のカフェデザイン。

## ターゲット

- **観光客**: 湘南・海岸エリアの観光で立ち寄る層
- **若い夫婦・カップル**: SNS映え意識
- **ノマド・コワーキング層**: 長居前提で WiFi・電源
- **地元の若者**: 西海岸カルチャーに憧れる層

## デザイン方針

- カラー: 海青（#2C6E91） × 砂生成（#F1E4D4） × 太陽イエロー（#F4A537） × チャコール（#1C2733）
- フォント: **Manrope**（欧文ヘッダー、Variable で太め）+ **Noto Sans JP**（和文）
- レイアウト: 標準グリッド型・8セクション
- ヒーロー: 全画面背景画像 + glass morphism stats カード
- 装飾要素: マーキー帯（"Since YYYY · ..."流れる）、メニューカード hover 時画像ズーム、ボタン shine effect

## 動的演出

- スクロール進捗バー（最上部、太陽×海グラデ）
- マーキー帯
- ヒーローパララックス
- セクション fade-up + ヒーロー数値カウントアップ
- メニューカード 3D tilt（hover）
- カーソルフォロワー（PC・hover対応端末のみ、太陽グロー）
- Story 背景の浮遊グロー（ふわっと動く）

## 画像差替え（実画像優先）

提案版では `placeholder-img` に CSS で配色だけ表示。実案件適用時は:

1. 媒体取材記事 / 公式SNS / 公式EC等から **店主公式の実画像** を取得
2. `images/real/` に保存
3. HTML の `<div class="placeholder-img">` を `<img src="images/real/{画像名}.jpg" alt="..." loading="lazy">` に置換
4. CSS の `.placeholder-img` ルールはそのまま残す（実画像が無いセクションは引き続き使う）

## 適合業種

- カフェ全般（特に湘南・海岸エリアの店舗）
- 特に「西海岸風」「オープンエア」「サーフ」「カリフォルニア」を訴求軸にできる店

## 不適合業種

- 純和食・割烹（→ cafe-editorial-magazine or 和系の bakery-storytelling-heritage を流用検討）
- 高級店（高級感が必要なら editorial-magazine 推奨）
- 整骨院・サロン系（業種別テンプレが整備され次第そちらを使用）
