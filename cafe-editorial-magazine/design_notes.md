# cafe-editorial-magazine 設計ノート

## コンセプト

**Editorial Magazine** — 雑誌の特集記事のような、見出し・キャプション・引用文・誌面番号を効かせたエディトリアルレイアウト。スプリット2カラムでテキスト密度を保ちつつ、大型タイポと余白で「読ませる」店舗HP。

## ターゲット

- **40代以上のグルメ感度高い層**: 雑誌・書籍に触れる文化感
- **メディア関係者・編集者**: 取材依頼の入口にもなる権威感
- **コーヒー専門誌・建築雑誌の読者**: 「品の良いカフェ」を探している層
- **老舗・歴史ある店の店主**: 「自分の店の格に合う」と感じやすい

## デザイン方針

- カラー: 黒（#0A0A0A） × 白（#FAF7F1） × 深い金（#B08D57） × アクセントレッド（#C8102E）
- フォント:
  - 欧文ヘッダー = **Playfair Display**（Variable / Italic 多用）
  - 和文 = **Noto Serif JP**（明朝・読み物）
  - UI = **Inter**（メタ情報・ナビ）
- レイアウト: スプリット2カラム + 雑誌風セクション番号
- ヒーロー: 左テキスト + 右大型カバー画像 + 数字キャプション付き
- 装飾要素: ドロップキャップ、引用文の大型クォートマーク、新聞風プレスリスト、誌面番号 "VOL.054"

## 動的演出（控えめ・editorial slow）

- スクロール進捗バー（赤）
- セクション fade-up（90ms間隔のスタガー、長め持続 0.9s）
- マーキー帯（黒地に金ドット、ゆったり 38秒）
- メニュー hover 時 行間 padding 拡張 + 背景クリーム化（紙の触感）
- access map に grayscale + コントラスト強化（誌面の白黒写真風）

## 画像差替え

`placeholder-hero` `placeholder-side` を実画像に置換。雑誌的な構図で:
- メイン画像: 大型・縦長（`hero-img-wrap` 内、aspect-ratio 4/5）
- サブ画像: 正方形を縦に2枚（`hero-side-col`）
- 各画像にキャプション "01 / The morning light, captured in dust." 形式

## 適合業種

- **老舗喫茶店**（純喫茶・1970年代以前創業）
- **コーヒー専門店**（焙煎所併設・スペシャルティコーヒー）
- **割烹×カフェ・茶寮**
- **書籍・古書併設のカフェ**
- **ギャラリー併設カフェ**（すずの木カフェ等が候補）
- **ホテルラウンジ**

## 不適合業種

- カジュアルカフェ・チェーン店（→ cafe-california-open-air 推奨）
- ファミリー向け（→ cafe-forest-hideaway 推奨）
