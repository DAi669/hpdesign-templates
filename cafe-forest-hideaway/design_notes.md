# cafe-forest-hideaway 設計ノート

## コンセプト

**Forest Hideaway** — 森の隠れ家・ゆったりとした時間が流れる小さなカフェ。葉のモチーフ、丸ゴシック、優しい曲線、Bento グリッドで「居心地の良さ」を視覚化。

## ターゲット

- **隠れ家カフェ好き**: 静かな時間を求める層
- **ファミリー連れ・ペット連れ**: 居心地・包容性重視
- **30-50代女性**: 落ち着いた店主との会話を楽しむ層
- **個人運営の店主**: 「私のお店の雰囲気に合う」と感じやすい

## デザイン方針

- カラー: 深緑（#2A4A37） × 苔色（#5C7C68） × 木目茶（#7A5C3F） × クリーム（#F5EDE0） × 薄ピンクアクセント（#E8C5C0）
- フォント:
  - 和文（メイン） = **Zen Maru Gothic**（柔らかい丸ゴシック）
  - 和文（タイトル） = **Noto Serif JP**（明朝・落ち着き）
  - 欧文 = **Cormorant Garamond**（繊細イタリック）
- レイアウト: 縦長一本道スクロール + Bento グリッド（メニュー）
- ヒーロー: 全画面背景画像 + 葉の浮遊アニメーション + 中央寄せメッセージ + scroll arrow
- 装飾要素:
  - 背景に SVG 葉パターン（fixed・透明度6%）
  - 浮遊する絵文字葉 🌿🍃（hero, cta-final）
  - ナビ hover 時に小さな🌿
  - signature-points 各行頭に 🌿
  - 大型 Bento グリッド（ある程度の不規則さ）

## 動的演出（ゆったり・有機的）

- スクロール時のヒーロー葉のパララックス（係数 0.1〜0.2）
- セクション fade-up 1.1秒 cubic-bezier (0.25, 0.46, 0.45, 0.94)（ゆったり）
- ヒーロー葉の継続 float アニメ（6秒周期）
- ヒーロー scroll arrow の上下バウンス
- About 数値カウントアップ（1.6秒）
- Bento カード hover で translateY + shadow

## 適合業種

- **隠れ家カフェ・1人運営の小さなカフェ**
- **ナチュラル・オーガニック系カフェ**
- **野菜中心のヘルシーカフェ**
- **店主のキャラクターが強い個人店**
- **ファミリー・ペット可カフェ**
- **ギャラリー併設カフェ**（cafe-editorial と競合、客層で選定）

## 不適合業種

- 高級店・権威系（→ cafe-editorial-magazine 推奨）
- 観光地・若年層・SNS映え系（→ cafe-california-open-air 推奨）

## 想定フィードバック（営業時の店主反応）

- 「うちの雰囲気にぴったり」
- 「お客様にゆったりしてもらいたいから、これが良い」
- 「ペット連れの常連さんが見ても違和感ない」

## 参考デザイントレンド

- 2026年版「ナチュラル・オーガニック」Webデザイン
- 余白多め、丸み、自然色、タイポは大きすぎず読みやすく
- Bento Grid の柔らかい派生（rounded corners, soft shadows）
- 葉・植物のモチーフをアクセント程度に（過剰使用を避ける）
